$\mathrm{SU}(N)$ Hubbard operators

TensorKitTensors.SUNHubbardOperators — Module
module SUNHubbardOperators

Module for the $SU(N)$ generalizations of the HubbardOperators operators.

Note

This module requires using SUNRepresentations in order to define the SUNIrrep-symmetric tensors.

TensorKitTensors.SUNHubbardOperators.hubbard_space — Method
hubbard_space(particle_symmetry::Type{<:Sector}, spin_symmetry::Type{<:Sector}; N::Integer = 3)

Return the local hilbert space for a Hubbard-type model with the given particle and spin symmetries. The basis is spanned by all possible fermionic color-like combinations of particles that satisfy the Pauli exclusion principle.

For example, for $SU(3)$ (N = 3), we have three colors (r, g and b), leading to the following eight possible states:

\[|0⟩, |r⟩, |g⟩, |b⟩, |rg⟩, |rb⟩, |gb⟩, |rgb⟩\]

Additionally, we can implement Trivial or U1Irrep symmetry for the particle number conservation.
